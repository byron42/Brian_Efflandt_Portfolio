Resume Analizer (Full Stack, AI)

AI‑Powered Resume Analyzer built with React, React Router, Puter, Puter.js, Tailwind CSS, Typescript, Zustand, and Vite. This is a streamlined platform where users can authenticate, upload resumes, and receive AI‑driven insights (including ATS scoring), job‑match analysis, and targeted improvement suggestions. This is all delivered through a clean, reusable UI.

ROBO FRIENDS (React)

Tic Tac Toe (React)

Expense Tracker (React)
